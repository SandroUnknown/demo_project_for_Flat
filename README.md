## <a name="Jenkins">**Jenkins**</a>

Тесты запускаются только через [Jenkins](https://jenkins.autotests.cloud/job/qa_homework_14_jenkins_crowdtesting_project/)

В качестве параметров можно выбрать:  
- Браузер
- Разрешение экрана
- Версию браузера
